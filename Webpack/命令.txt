

*全局下编译打包
webpack entry.js output.js
非全局下编译打包
node_modules/.bin/webpack entry.js output.js

*配置文件webpack.config.js的entry、output后可以省略输入以上地址
全局下编译打包
webpack
非全局下编译打包
node_modules/.bin/webpack

*配置package.json："scripts": {
    "start": "webpack"
  }
直接运行 cnpm run start 或 cnpm start
就可以直接编译打包

*使用webpack构建本地服务器，实时刷新：
安装
cnpm install webpack-dev-server --save-dev 
配置文件
webpack.config.js添加devServer选项
package.json 的scripts添加sever选项
运行cnpm run server

*配置loader:
安装babel-loader
$ cnpm install --save-dev babel-core babel-loader babel-preset-es2015 babel-preset-stage-2
安装style-loader、css-loader
$ cnpm install --save-dev style-loader css-loader

*插件
$ cnpm install --save-dev hot-module-replacement-plugin
