object-fit用于控制“替换元素”（img,vedio,iframe...具有src属性的元素）的内容样式的，替换元素分为壳（容器）与替换内容，object-fit控制的永远是替换内容的尺寸表现

object-fit: fill（默认）

object-fit: contain（不变形、不裁剪、去适应最大的边）

object-fit: contain（不变形、不裁剪、去适应最大的边）

object-fit: cover（裁剪、居中、去适应最小的边）

object-fit: cover（裁剪、居中、去适应最小的边）

object-fit: none（保持原始大小、居中）

object-fit: none（保持原始大小、居中）

object-fit: scale-down（具有动态特性）

object-fit: scale-down（具有动态特性，兼有contain、none的特性）

object-position是控制替换内容位置的。默认值是50% 50%，也就是居中效果。 支持left right top bottom作为参数
